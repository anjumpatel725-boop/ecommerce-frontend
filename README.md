# 🛍️ E-Commerce Frontend

A responsive E-Commerce web application built with **React.js** and **Vite**, providing a modern shopping experience with user authentication, cart management, checkout, and an admin dashboard.

## 🚀 Features

- User Login & Registration
- Product Listing & Search
- Shopping Cart
- Checkout
- Order History
- Admin Dashboard
- Product & Order Management
- Responsive Design

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript
- Axios
- React Router
- CSS
- Recharts

## 📦 Installation

```bash
git clone https://github.com/anjumpatel725-boop/ecommerce-frontend.git
cd ecommerce-frontend
npm install
npm run dev
```
